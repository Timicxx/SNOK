# SNOK
### A snake game that uses an arduino, joystick and a LED Matrix.

*Hardware required:*
- Arduino UNO
- MAX7219 Led Matrix
- Joystick module

*Connection:*
```
LED MATRIX CONNECTION:
DIN   =   DIGITAL  12
CLK   =   DIGITAL  11
CS    =   DIGITAL  10

JOYSTICK CONNECTION:
SW    =   DIGITAL  2
X     =   ANALOG   0
Y     =   ANALOG   1
```
